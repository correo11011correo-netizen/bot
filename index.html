Política de Privacidad

Última actualización: 22 de diciembre de 2025

Quiénes somos

Esta aplicación/bot opera sobre las APIs de Meta (WhatsApp, Messenger e Instagram) para enviar y recibir mensajes.

Contacto: contacto@tudominio.com

Datos que procesamos

  • Identificadores de cuenta: ID del remitente, número de teléfono o usuario.
  • Contenido del mensaje: texto y metadatos necesarios para responder.
  • Logs técnicos: fecha/hora, estado de entrega, errores.

Finalidad del uso

Usamos los datos exclusivamente para operar el bot, responder consultas y mejorar la experiencia del usuario.

Base legal

Procesamos datos para ejecutar el servicio solicitado por el usuario y cumplir términos de las plataformas de Meta.

Conservación

Retenemos los datos el tiempo mínimo necesario para prestar el servicio y cumplir obligaciones legales.

Compartición

No vendemos ni compartimos datos personales con terceros, salvo proveedores necesarios para la operación (hosting, logs) y exigencias legales.

Derechos de los usuarios

  • Acceso y rectificación: podés pedir copia o corregir tus datos.
  • Eliminación: podés solicitar borrar tus datos cuando corresponda.
  • Contacto: escribí a contacto@tudominio.com.

Seguridad

Aplicamos medidas razonables para proteger los datos contra acceso no autorizado, alteración o divulgación.

Actualizaciones

Podemos actualizar esta política. La versión vigente siempre estará publicada en esta página.
